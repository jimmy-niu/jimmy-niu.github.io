Maps (Java / Spark Backend, Web Frontend)

Maps was a partner project that took latitude and longitude pairs, street names, and lots of finger flexing to create a visual map and route planner. Our final product allowed users to select start/end location via clicking on the map or typing into autocorrected fields, and then showed them the shortest path using A* search.
